# AquaVenture Web Application

AquaVenture เป็นเว็บแอปพลิเคชันสำหรับระบบกิจกรรมท่องเที่ยวทางน้ำ เช่น ดำน้ำ, เจ็ตสกี, ล่องเรือ และดำน้ำตื้น  
โปรเจกต์นี้พัฒนาเพื่อใช้ในการเรียนรู้และฝึกพัฒนา Web Application แบบ Full Stack โดยแยกส่วน Frontend และ Backend 

---

## ฟีเจอร์หลักของระบบ

### ผู้ใช้งาน (Customer)
- สมัครสมาชิก / เข้าสู่ระบบ
- ดูกิจกรรมทางน้ำ (Scuba, Snorkeling, Jet Ski, Boat)
- ดูรายละเอียดกิจกรรม
- จองกิจกรรม
- ดูข้อมูลการจอง
- หน้าขอบคุณหลังทำรายการ

### ผู้ให้บริการ (Provider)
- Dashboard สำหรับผู้ให้บริการ
- ดูรายการจองของลูกค้า
